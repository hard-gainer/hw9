# Задание

В файле students.csv дана таблица.
Необходимо разработать модель данных для хранения данных из этой таблицы.

## Этапы работы:

1. Опишите модель данных, используя SQLAlchemy.

2. Разработайте класс для выполнения операций INSERT и SELECT для полученной модели данных.

3. Напишите метод для заполнения модели данными из файла students.csv.

4. Напишите методы для:
- получения списка студентов по названию факультета;
- получения списка уникальных курсов;
- получения среднего балла по факультету;
- получения списка студентов по выбранному курсу с оценкой ниже 30 баллов.
